HTML Form Reference

Form Tags

Tag Name Description
<form> Form Defines a data entry area that contains input elements
<input> Input Creates an input field in which data can be entered
<label> Label Creates a label or brief description before input elements
<textarea> Text Area Creates a multi-line text input control
<select> Select Creates a dropdown control
<button> Button Creates a clickable button that can contain text or an image
<datalist> Data List Specifies a list of options for a textfield (<input>) element
<fieldset> Field Set Groups related form elements and displays a box with a legend around these
<legend> Legend Defines a caption for a fieldset

Form Attributes

Attribute Name Description
accept-charset Accept Charset Specifies the character encodings used for form submission
action Action Specifies where to send the form-data when a form is submitted
autocomplete Auto Complete Specifies whether a form should have autocomplete on or off
enctype Encryption Type Specifies how the form-data should be encoded when submitting it to the server (only for method="post")
method Method Specifies the HTTP method to use when sending form-data
name Name Specifies the name of the form
novalidate Not Validate Specifies that the form should not be validated when submitted
rel Relation Specifies the relationship between a linked resource and the current document
target Target Specifies where to display the response that is received after submitting the form
